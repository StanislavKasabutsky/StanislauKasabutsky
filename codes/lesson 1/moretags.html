термины и описания к ним <DL> <DT> <DD>

<dl>
  <dt>термин </dt>
  <dd>описание или определение</dd>

данные теги предназначены для нумерации терминов и пояснений к ним:

Подлежащее
главный член предложения, обозначающий предмет, к-рому приписывается действие или признак, названные в сказуемом.
Сказуемое
главный член предложения, обозначающий действие или состояние предмета, выраженного подлежащим.

описание кода через <CODE>

данный тег используется для своеобразного выделения кода на странице (моноширинный текст уменьшенного размера), например:


____________________________________________________________________________________
Нечем заняться? Напиши простой код для перехода по ссылке на вики Бенито Муссолини:
____________________________________________________________________________________ 

        #include 
            #include 
             
            int main(int argc, char * argv [])
            {
                system("explorer https://ru.wikipedia.org/wiki/Муссолини,_Бенито");
             
                return 0;
            }
_______________________________________________________________________________________________

также cюда можно отнести и тег <tt> который делает буквально то же самое:


различий визуально абсолютно никаких, вообще.
 


предназначение <KBD>

Данный тег указывает на то, что текст внутри элемента описывает пользовательский ввод с клавиатуры, голосовой ввод или ввод с использованием любого другого типа устройств ввода текста.
